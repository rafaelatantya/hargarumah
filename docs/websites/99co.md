## 1. Overview

99.co is a major Southeast Asian property portal. Same parent group as Rumah123 (REA Group). Covers houses, apartments, land, and commercial properties. The search results page for Bekasi Selatan shows **~432 listings** with **~20 listings per page** (~22 pages total).

## 2. URL Patterns

### Search URL
```
https://www.99.co/id/jual/rumah/{city}/{district}
```

### URL Parameters
| Parameter | Description | Example |
|---|---|---|
| `hlmn` | Page number (1-indexed) | `?hlmn=2` |

**Note**: Pagination uses `hlmn` (short for "halaman" = page in Indonesian), NOT `page`.

### Example URLs
```
# Page 1 (default)
https://www.99.co/id/jual/rumah/bekasi/bekasi-selatan

# Page 2
https://www.99.co/id/jual/rumah/bekasi/bekasi-selatan?hlmn=2

# Other districts
https://www.99.co/id/jual/rumah/bekasi/bekasi-barat
https://www.99.co/id/jual/rumah/bekasi/rawalumbu
https://www.99.co/id/jual/rumah/bekasi/bekasi-timur
```

## 3. Search Flow (Step by Step)

1. Navigate directly to `https://www.99.co/id/jual/rumah/bekasi/{district}`
2. Page loads with search results immediately — no interaction needed
3. Location is encoded in the URL path
4. Results show a count header (e.g., "432 rumah dijual di Bekasi Selatan")
5. Listing cards displayed in vertical list format
6. Mix of regular listings and "Project" (developer) listings
7. Scroll down to see all listings on current page
8. Pagination bar at bottom with page numbers and next arrow
9. **Gotcha**: A "Helper Bubble" tutorial overlay may appear on page 2+ — must close it by clicking "Tutup"

## 7. Edge Cases & Gotchas

- **"Helper Bubble" overlay**: On page 2+, a tutorial overlay ("Anda dapat mengatur pencarian...") may appear blocking interaction. Must click "Tutup" (close) button to dismiss. The scraper should handle this.
- **Project/Developer listings**: Mixed in with regular listings. URLs contain `/projects/` instead of `/properti/`. Filter by checking if URL contains `/properti/`.
- **Dynamic count**: Total listing count may change between pages (e.g., 432 on page 1 → 407 on page 2). Use the first page count for reference.
- **Same parent as Rumah123**: May share similar anti-bot infrastructure. Don't scrape both simultaneously.
